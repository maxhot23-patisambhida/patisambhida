# assets

ไฟล์ตกแต่งของเว็บ (ไม่ใช่รูปประกอบเนื้อหา — รูปประกอบ editorial อยู่ที่ `web/editorial-images/`)

## hero-stupa.png — ลายน้ำหลัง hero ห้องสมุด

ภาพพระธาตุ (พระธาตุพนม ยามอาทิตย์อัสดง) ใช้เป็น **ลายน้ำจางๆ** หลัง hero บนหน้า `#/library`

- ไฟล์ปัจจุบัน **`web/assets/hero-stupa.png`** (ชื่อ/นามสกุลต้องตรงกับ path ใน `.hero-bg`)
- ระบบครอป/เบลอ/ลดความทึบให้เองด้วย CSS (`.hero-bg` ใน `styles.css`) — ไม่ต้องแก้ไฟล์ภาพ
  - ครอป: `background-position: center 28%` (โชว์ยอดพระธาตุ + ดวงอาทิตย์ + ท้องฟ้า)
  - จาง: `opacity 0.12` (กลางคืน 0.10) · เบลอ `blur(2px)` · จางหายลงล่างด้วย mask
- ถ้ายังไม่วางไฟล์ hero จะแสดงพื้นกระดาษงาช้างล้วน (ไม่มี broken image เพราะเป็น CSS background)
- ถ้าเปลี่ยนนามสกุลไฟล์ (เช่น `.webp` เล็กกว่า) ต้องแก้ path ใน `.hero-bg { background-image }` ให้ตรงด้วย — มิฉะนั้นจะ 404 และลายน้ำไม่ขึ้น
