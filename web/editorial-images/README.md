# editorial-images

รูปภาพประกอบ (แผนภาพหมวดธรรม ฯลฯ) ที่แทรกในเนื้อหาผ่านชั้น **Editorial → image-block**

- วางไฟล์ภาพ (`.webp` / `.png` / `.jpg` / `.svg`) ไว้ในโฟลเดอร์นี้
- อ้างอิงในไดอะล็อก "🖼 แทรกรูปภาพ" ด้วย path เช่น `/editorial-images/khandha-5.webp`
- รองรับ URL ภายนอก (`https://…`) ได้เช่นกัน

`khandha-5.svg` = ตัวอย่างแผนภาพขันธ์ ๕ (ใช้ทดสอบ/เป็นแม่แบบ)
