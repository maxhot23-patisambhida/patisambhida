# ux/ — เอกสารด้านประสบการณ์ผู้ใช้

โฟลเดอร์นี้บรรจุเอกสารที่ขยายความด้าน **ประสบการณ์การศึกษา** — หลักการอ่าน การนำทาง ความเรียบง่าย และแนวทาง Mobile First

- ยังคงอยู่ในระดับหลักการ จึง **ไม่กล่าวถึงเทคโนโลยีหรือรายละเอียดการ Implement**
- ต้องสอดคล้องกับ [PKA-STD-007 — มาตรฐานประสบการณ์ผู้ใช้](../standards/PKA-STD-007-มาตรฐานประสบการณ์ผู้ใช้.md)
- ทุกฉบับต้องอ้างอิง [PKA-STD-001 (เอกสารแม่)](../standards/PKA-STD-001-ธรรมนูญโครงการ.md)

**สถานะ:** ยังไม่มีเอกสาร (Planned)
