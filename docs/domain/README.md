# domain/ — แบบจำลองโดเมน (PKA-DOM) — Level 3

โฟลเดอร์นี้บรรจุเอกสารตระกูล **แบบจำลองโดเมน (`PKA-DOM-*`)** ซึ่งเป็นเอกสาร **ระดับที่ 3** ของลำดับชั้นเอกสารโครงการ

## หน้าที่ของตระกูล DOM

- ทำหน้าที่ **กำหนดแบบจำลองเชิงแนวคิดขององค์ความรู้** (Conceptual Model) — ชนิดขององค์ความรู้ ชนิดของความสัมพันธ์ และโครงสร้างมาตรฐานของหน่วยองค์ความรู้
- **ไม่ใช่องค์ความรู้จริง** — เนื้อหาองค์ความรู้ที่แท้จริงเป็นเรื่องของชั้น Knowledge (Level 5, โฟลเดอร์ `knowledge/`)
- **ไม่ใช่สถาปัตยกรรม** — ภาพรวมของระบบเป็นเรื่องของชั้น Architecture (Level 2)
- **ไม่ใช่มาตรฐาน** — กฎกลางเป็นเรื่องของชั้น Standards (Level 0–1)

## ตำแหน่งในลำดับชั้น

```
   Level 2  สถาปัตยกรรม (Architecture)
        ▼
   Level 3  แบบจำลองโดเมน (Domain)   ← โฟลเดอร์นี้
        ▼
   Level 4  แหล่งอ้างอิง (Reference)
```

ตระกูล DOM อยู่ **ระหว่าง Architecture และ Reference** — รับกรอบภาพรวมจากสถาปัตยกรรม และวางแบบจำลองให้ชั้นองค์ความรู้จริงนำไปใช้

- ยังคงอยู่ในระดับแนวคิด จึง **ไม่กล่าวถึงเทคโนโลยีหรือรายละเอียดการ Implement**
- ออกแบบให้ **รองรับการขยายแบบจำลองโดเมนในอนาคต** โดยไม่ต้องเปลี่ยนสถาปัตยกรรมหลัก
- ทุกฉบับต้องอ้างอิง [PKA-STD-001 (เอกสารแม่)](../standards/PKA-STD-001-ธรรมนูญโครงการ.md) และสอดคล้องกับ [PKA-STD-002](../standards/PKA-STD-002-มาตรฐานข้อมูล.md), [PKA-STD-004](../standards/PKA-STD-004-มาตรฐานพระธรรมและลำดับชั้นแหล่งข้อมูล.md)
