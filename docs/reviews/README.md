# reviews/ — เอกสารประเมินสถาปัตยกรรม (PKA-REV)

โฟลเดอร์นี้บรรจุเอกสารตระกูล **เอกสารประเมิน (`PKA-REV-*`)** ซึ่งทำหน้าที่ **ทบทวนและประเมินสถาปัตยกรรมของโครงการ**

## หน้าที่ของตระกูล REV

- บันทึก **ผลการทบทวน/ประเมิน** สถาปัตยกรรม เอกสาร และการกำกับดูแลของโครงการ
- เป็นเอกสาร **ขวางทุกระดับ (Cross-cutting)** เช่นเดียวกับบันทึกการตัดสินใจ (ADR) — ประเมินได้ทุกชั้นตั้งแต่ Vision ถึง Knowledge
- **ไม่ใช่มาตรฐาน (STD), ไม่ใช่สถาปัตยกรรม (ARC), ไม่ใช่แบบจำลองโดเมน (DOM)** — แต่ *ประเมิน* เอกสารเหล่านั้น
- ยังคงอยู่ในระดับแนวคิด จึง **ไม่กล่าวถึงเทคโนโลยีหรือรายละเอียดการ Implement**
- ทุกฉบับต้องอ้างอิง [PKA-STD-001 (เอกสารแม่)](../standards/PKA-STD-001-ธรรมนูญโครงการ.md)
