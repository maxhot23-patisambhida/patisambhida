# decisions/ — บันทึกการตัดสินใจเชิงสถาปัตยกรรม (ADR)

โฟลเดอร์นี้บรรจุ **Architecture Decision Records (ADR)** (ระดับ 6) คำนำหน้า `ADR-NNNN`

- ADR แต่ละฉบับบันทึก **เหตุผล** ของการเลือกแนวทางหนึ่งเหนืออีกแนวทาง พร้อมบริบทและผลที่ตามมา
- หมายเลขเรียงตามลำดับการตัดสินใจ (4 หลัก เริ่มที่ `0001`) ห้ามใช้หมายเลขซ้ำ
- ADR ที่ถูกแทนที่ให้คงไว้และเปลี่ยนสถานะเป็น *Superseded* พร้อมระบุฉบับที่มาแทน
- ทุกฉบับต้องสอดคล้องกับ [PKA-STD-001 (เอกสารแม่)](../standards/PKA-STD-001-ธรรมนูญโครงการ.md) และมาตรฐานที่เกี่ยวข้อง

**สถานะ:** ยังไม่มีเอกสาร (Planned) — สร้างจาก [`../templates/ADR-TEMPLATE.md`](../templates/ADR-TEMPLATE.md)
