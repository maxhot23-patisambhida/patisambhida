# standards/ — เอกสารมาตรฐาน (PKA-STD)

โฟลเดอร์นี้บรรจุเอกสารมาตรฐานของโครงการ คำนำหน้า `PKA-STD-NNN`

- **เอกสารแม่:** [PKA-STD-001 — ธรรมนูญโครงการ](PKA-STD-001-ธรรมนูญโครงการ.md) (ระดับ 0) เป็นที่มาของหลักการทั้งหมด
- เอกสารมาตรฐานอื่น (ระดับ 1) ขยายความหลักการของเอกสารแม่เป็นมาตรฐานเฉพาะด้าน
- ทุกฉบับต้องอ้างอิงเอกสารแม่และไม่ขัดแย้งกับเอกสารแม่
