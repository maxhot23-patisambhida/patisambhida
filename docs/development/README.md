# development/ — คู่มือปฏิบัติ (GUIDE)

โฟลเดอร์นี้บรรจุ **คู่มือปฏิบัติ (Guides)** (ชั้นปฏิบัติ — อยู่ใต้ระดับเอกสารแนวคิดทั้งเจ็ด) คำนำหน้า `GUIDE-NNN`

- คู่มือให้แนวทางการทำงานและการปฏิบัติจริง โดยอยู่ภายใต้มาตรฐานและสถาปัตยกรรมของโครงการ
- หมายเลข 3 หลัก เริ่มที่ `001` ห้ามใช้หมายเลขซ้ำ
- ต้องสอดคล้องกับ [PKA-STD-008 — มาตรฐานการพัฒนาและการกำกับเอกสาร](../standards/PKA-STD-008-มาตรฐานการพัฒนาและการกำกับเอกสาร.md)
- ทุกฉบับต้องอ้างอิง [PKA-STD-001 (เอกสารแม่)](../standards/PKA-STD-001-ธรรมนูญโครงการ.md)

**สถานะ:** ยังไม่มีเอกสาร (Planned)
