# agents/catalog/ — สารบบตัวแทนงาน (Agent Catalog)

โฟลเดอร์นี้เป็น **สารบบตัวแทนงานรายตัว (Agent Catalog)** — ที่รวมตัวแทนงานแต่ละตัวที่กรอก **พิมพ์เขียวตัวแทนงาน** ครบทุกองค์ประกอบตาม [PKA-AGT-002](../PKA-AGT-002-พิมพ์เขียวตัวแทนงาน.md)

## แม่แบบของสารบบ (Catalog Template)

| รหัส | ชื่อ | บทบาท | สถานะเอกสาร |
|---|---|---|---|
| [PKA-AGT-CAT-000](PKA-AGT-CAT-000-ต้นแบบมาตรฐานตัวแทนงาน.md) | ต้นแบบมาตรฐานตัวแทนงาน (Canonical Agent Template) | แม่แบบกลางของตัวแทนงานทุกชนิด — ไม่ใช่ตัวแทนงานจริง | Draft 0.1 |
| [PKA-AGT-CAT-001A](PKA-AGT-CAT-001A-มาตรฐานผู้อำนวยการสำนัก.md) | มาตรฐานผู้อำนวยการสำนัก (Office Director Canonical Standard) | แม่แบบเฉพาะของผู้อำนวยการสำนัก (CAT-002–012) — ไม่ใช่ตัวแทนงานจริง | Draft 0.1 |

> หมายเลข `000` **สงวนไว้สำหรับแม่แบบ** — ตัวแทนงานจริงเริ่มที่ `001` ตัวแทนงานทุกตัวต้องสืบทอดโครงสร้างจาก [PKA-AGT-CAT-000](PKA-AGT-CAT-000-ต้นแบบมาตรฐานตัวแทนงาน.md)
>
> [PKA-AGT-CAT-001A](PKA-AGT-CAT-001A-มาตรฐานผู้อำนวยการสำนัก.md) เป็น **มาตรฐานกลางเฉพาะของบทบาทผู้อำนวยการสำนัก** (ไม่ใช่ตัวแทนงานจริง) สืบทอดจาก CAT-000 และเป็นแม่แบบที่ผู้อำนวยการรายสำนักทุกคน (CAT-002–012) ต้องยึด

## ทะเบียนตัวแทนงาน (Agent Catalog Registry)

| รหัส | ชื่อ | ระดับ | สถานะวงจรชีวิต | สถานะเอกสาร |
|---|---|---|---|---|
| [PKA-AGT-CAT-001](PKA-AGT-CAT-001-Chief-Knowledge-Orchestrator.md) | ผู้บริหารงานองค์กร (Chief Knowledge Orchestrator — CKO) | 2 | ออกแบบแล้ว (Designed) | Draft 0.1 |

> **PKA-AGT-CAT-001 (CKO)** เป็นตัวแทนงานจริงตัวแรกของสารบบ และเป็น **ตัวอย่างคุณภาพระดับ Gold Standard** ที่กรอกแม่แบบ CAT-000 ครบทุกองค์ประกอบ

## การเพิ่มตัวแทนงานใหม่

1. ใช้รหัส `PKA-AGT-CAT-NNN` (3 หลัก เริ่ม `001`; `000` สงวนไว้สำหรับแม่แบบ) — ห้ามใช้หมายเลขซ้ำ
2. คัดลอกโครงสร้างจาก [PKA-AGT-CAT-000 — ต้นแบบมาตรฐาน](PKA-AGT-CAT-000-ต้นแบบมาตรฐานตัวแทนงาน.md) และดู [PKA-AGT-CAT-001 (CKO)](PKA-AGT-CAT-001-Chief-Knowledge-Orchestrator.md) เป็นตัวอย่างที่กรอกครบ; เติมข้อมูลตาม [PKA-AGT-002 — พิมพ์เขียวตัวแทนงาน](../PKA-AGT-002-พิมพ์เขียวตัวแทนงาน.md#13-แม่แบบเปล่าของพิมพ์เขียว-blank-template) ให้ครบ หากด้านใดไม่มีให้ระบุ "ไม่มี" พร้อมเหตุผล — **หากเป็นผู้อำนวยการสำนัก (CAT-002–012) ต้องยึดมาตรฐานเพิ่มเติมจาก [PKA-AGT-CAT-001A — มาตรฐานผู้อำนวยการสำนัก](PKA-AGT-CAT-001A-มาตรฐานผู้อำนวยการสำนัก.md) และผ่านรายการตรวจสอบคุณภาพในหัวข้อ 16 ของเอกสารนั้น**
3. ตรวจให้สอดคล้องกับ [PKA-AGT-001](../PKA-AGT-001-ธรรมนูญตัวแทนงาน.md), [PKA-AGT-003](../PKA-AGT-003-มาตรฐานบทบาทและความรับผิดชอบ.md), [PKA-AGT-004](../PKA-AGT-004-มาตรฐานการสื่อสารระหว่างตัวแทนงาน.md) และ [PKA-AGT-006](../PKA-AGT-006-แบบจำลองการทำงานร่วมกันของตัวแทนงาน.md)
4. ระบุสถานะวงจรชีวิตตาม [PKA-AGT-005](../PKA-AGT-005-วงจรชีวิตตัวแทนงาน.md)
