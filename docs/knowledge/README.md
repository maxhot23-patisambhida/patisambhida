# knowledge/ — องค์ความรู้จริง (Knowledge Assets) — Level 5

โฟลเดอร์นี้ **สงวนไว้สำหรับชั้นที่ 5 — องค์ความรู้จริง (Knowledge Assets)**

## หน้าที่ของโฟลเดอร์

- เก็บ **องค์ความรู้จริง** ที่ถูกจัดโครงสร้างตามแบบจำลองโดเมน (Level 3) และอ้างอิงแหล่งอ้างอิง (Level 4)
- ในอนาคตจะบรรจุเฉพาะเอกสารตระกูล **`PKA-KNW-xxxx`** เท่านั้น
- ยังคงอยู่ในระดับแนวคิด จึง **ไม่กล่าวถึงเทคโนโลยีหรือรายละเอียดการ Implement**
- ทุกฉบับต้องอ้างอิง [PKA-STD-001 (เอกสารแม่)](../standards/PKA-STD-001-ธรรมนูญโครงการ.md)

## ตำแหน่งในลำดับชั้น

```
   Level 4  แหล่งอ้างอิง (Reference)
        ▼
   Level 5  องค์ความรู้จริง (Knowledge)   ← โฟลเดอร์นี้
        ▼
   Level 6  บันทึกการตัดสินใจ (Decision Records)
```

## สถานะ

**ยังไม่มีเอกสารในเฟสปัจจุบัน (Reserved)** — โฟลเดอร์นี้จะได้รับการจัดโครงสร้างเมื่อเข้าสู่เฟสองค์ความรู้จริง
