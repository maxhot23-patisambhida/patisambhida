# reference/ — แหล่งอ้างอิง (PKA-REF)

โฟลเดอร์นี้บรรจุเอกสารตระกูล **แหล่งอ้างอิง (`PKA-REF-*`)** (ระดับ 4) ซึ่งอธิบาย **คัมภีร์และแหล่งอ้างอิงต้นทาง** ตามลำดับชั้นแหล่งข้อมูล

- หน้าที่ของตระกูล REF คือ *อธิบายแหล่งอ้างอิง* — ไม่ใช่กำหนดกฎ (STD), ไม่ใช่อธิบายภาพรวม (ARC), ไม่ใช่แบบจำลอง (DOM) และไม่ใช่เก็บองค์ความรู้จริง (KNW)
- ยังคงอยู่ในระดับแนวคิด จึง **ไม่กล่าวถึงเทคโนโลยีหรือรายละเอียดการ Implement**
- ออกแบบให้รองรับการขยายไปยัง **พระไตรปิฎก อรรถกถา ฎีกา และคัมภีร์อื่น** ในอนาคต
- ต้องสอดคล้องกับ [PKA-STD-004 — มาตรฐานพระธรรมและลำดับชั้นแหล่งข้อมูล](../standards/PKA-STD-004-มาตรฐานพระธรรมและลำดับชั้นแหล่งข้อมูล.md) และ [PKA-STD-003 — มาตรฐานการอ้างอิง](../standards/PKA-STD-003-มาตรฐานการอ้างอิง.md)
- ทุกฉบับต้องอ้างอิง [PKA-STD-001 (เอกสารแม่)](../standards/PKA-STD-001-ธรรมนูญโครงการ.md)
