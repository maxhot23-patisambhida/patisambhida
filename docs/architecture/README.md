# architecture/ — เอกสารสถาปัตยกรรม (PKA-ARC)

โฟลเดอร์นี้บรรจุเอกสารตระกูล **สถาปัตยกรรม (`PKA-ARC-*`)** (ระดับ 2) ซึ่งอธิบาย **ภาพรวมของระบบและการเชื่อมโยง**

## หน้าที่ของตระกูล ARC

- ตระกูล ARC อธิบาย **"ภาพรวม"** ของโครงการ — โครงการประกอบด้วยชั้นใด แต่ละชั้นทำหน้าที่อะไร และเชื่อมโยงกันอย่างไร
- ตระกูล ARC อธิบาย *ภาพรวม* **ไม่ใช่ "วิธีพัฒนา"** และ **ไม่กล่าวถึงเทคโนโลยีใด ๆ**

## ความสัมพันธ์กับ STD และ DOM

- **อยู่ภายใต้ STD** — สถาปัตยกรรมอธิบายภาพรวมตามกรอบของมาตรฐาน โดย **ไม่กำหนดกฎเอง** (การกำหนดกฎเป็นหน้าที่ของตระกูล STD)
- **กำกับภาพรวมให้ DOM** — แบบจำลองโดเมน (DOM) ทำงานภายในภาพรวมที่สถาปัตยกรรมวางไว้ โดย ARC **ไม่กำหนดแบบจำลองเอง** (เป็นหน้าที่ของตระกูล DOM)
- **ข้อบังคับการแยกหน้าที่:** มาตรฐาน (STD) และแบบจำลองโดเมน (DOM) **ต้องไม่ถูกย้ายเข้ามาเป็นสถาปัตยกรรม (ARC)**
- ทุกฉบับต้องอ้างอิง [PKA-STD-001 (เอกสารแม่)](../standards/PKA-STD-001-ธรรมนูญโครงการ.md)
